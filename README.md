## 📱 Responsividade

O portfólio foi desenvolvido com abordagem **mobile-first** e é totalmente responsivo:

- **Desktop** (1200px+): Layout completo com 2-3 colunas
- **Tablet** (768px - 1199px): Layout adaptado com 1-2 colunas  
- **Mobile** (até 767px): Layout em coluna única com menu hamburger
